🎥 Video to Text Summarizer using NLP (Advanced)

📌 Overview

An AI-powered application that converts videos into transcripts and concise summaries using Whisper and NLP models.

🚀 Features

- 🎥 Upload video
- 🔊 Audio extraction
- 🧠 Speech-to-text using OpenAI Whisper
- ✂️ Text summarization using BART
- 📄 Download transcript
- 📝 Download summary
- ⚡ Handles long videos (chunking)

🛠 Tech Stack

- Python
- Streamlit
- OpenAI Whisper
- Hugging Face Transformers (BART)
- MoviePy

▶️ Run Locally

pip install -r requirements.txt
streamlit run app.py

📂 Project Files

- app.py → main application
- requirements.txt → dependencies
- project-report.pdf → documentation
